RejectWorkOrderQuoteRequest

    Description

      The request supports rejecting of Work Order Quote Items, Squawks, Steps, Parts, Services and Misc Charges..

    Preconditions

      Work order quote entity must be in a valid state to be rejectd. The "Rejected by" user must have proper permissions.
