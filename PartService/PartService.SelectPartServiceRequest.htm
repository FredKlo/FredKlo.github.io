SelectPartServiceRequest

    Description

      This request selects a part/service request in the Part/Service Request module.

    Preconditions

      User must have proper permissions to select parts and services.

      		X

      LineItemNumber Integer

      PurchaseRequestFormattedNumber String  
      		255
      		*
      		X
      		Either/or relationship with LineItemNumber / PurchaseRequestMasterNumber.
